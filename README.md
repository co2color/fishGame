# vue-project

small games by vue3+ts+vite+tw


## Project Setup

```sh
npm i -g pnpm ( if you have pnpm, skip this one)
pnpm i
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
